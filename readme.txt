ciao, questa è una calcolatrice.
requisiti per eseguire questi tipi di file:
- un pc;
- node(che si può scaricare su: https://nodejs.org/it/
- un terminale.
per eseguire i file bisogna recarsi nella\ndirectory contenente il file calcolatrice.js usando il comando cd,
una volta che si è nella directory giusta, bisogna eseguire il file usando il comando: node calcolatrice.js
ecco gli argomenti che dovrete passare:
- un numero qualsiasi;\n
- un segno che in informatica sono:
"+ = addizione;
- = sottrazione;
* = moltiplicazione;
/ = divisione intera;
% = divisione con resto.
- un secondo numero;
esempio: node calcolatrice.js 1 + 1
